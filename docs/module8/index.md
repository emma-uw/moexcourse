# Модуль 8: Риск-менеджмент и Запуск (Production)

Последний модуль курса посвящен подготовке торгового робота к реальной работе. Мы научимся управлять рисками, мониторить работу и развертывать систему в продакшн.

## Цель

Сделать робота безопасным и автономным. Обеспечить надежную работу 24/7 с контролем рисков и оперативным мониторингом.

## Разделы

- [8.1. Жесткие риски](./8-1-risks.md): Stop-Loss и Take-Profit в коде. Ограничение потерь на день (Daily Loss Limit). Кнопка "Kill Switch" (экстренная остановка).
- [8.2. Логирование](./8-2-logging.md): Библиотека logging. Записываем каждое "чихание" робота в файл для разбора полетов.
- [8.3. Уведомления](./8-3-notifications.md): Подключение Telegram-бота для отчетов на телефон.
- [8.4. Деплой](./8-4-deploy.md): Запуск робота на облачном сервере (VPS), чтобы он работал 24/7, даже когда ваш ноутбук выключен.

## Результат модуля

Финальный проект — автономный торговый робот на виртуальном сервере с уведомлениями в Telegram.
