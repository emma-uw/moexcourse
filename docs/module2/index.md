# Модуль 2: Погружение в данные MOEX AlgoPack

Второй модуль курса посвящен глубокому изучению рыночных данных, доступных через MOEX AlgoPack. Мы научимся анализировать не только цены, но и другие важные показатели, которые дают представление о поведении участников рынка.

![image](/img/datashop.png)

## Цель

Научиться читать "рентген" рынка, а не только цены. Понять, как инициаторы сделок, дисбаланс в стакане и другие метрики влияют на движение цены.

## Разделы

- [2.1. Свечи, Сделки, Стакан](./2-1-candles.md): Понятие инициатора сделки (кто бьет по рынку: покупатель или продавец?).
- [2.2. SuperCandles](./2-2-supercandles.md): Производные данные
- [2.3. FUTOI и HHI](./2-3-futoi-hhi.md): Уникальная торговая аналитика

## Результат модуля

Исследовательский ноутбук, где на пример на графике цены отмечены моменты аномального дисбаланса в стакане.

Проблема: аномальный дисбаланс в стакане будет виден только в конце торгового дня?
