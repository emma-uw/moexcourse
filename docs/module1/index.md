# Модуль 1: Фундамент и Инструментарий

Первый модуль курса посвящен созданию фундамента для работы с алгоритмической торговлей. Мы начнем с настройки рабочей среды, изучим основные инструменты для работы с данными и познакомимся с API Московской биржи.

## Цель

Подготовить рабочее место и освоить базовый синтаксис для работы с финансовыми данными. Научиться загружать и визуализировать рыночную информацию.

## Разделы

- [1.1. Настройка среды трейдера](./1-1-setup.md): Установка Anaconda/Python, Jupyter Lab
- [1.2. Основы Pandas](./1-2-pandas.md): Ресемплинг (превращение минутных свечей в часовые)
- [1.3. Визуализация](./1-3-visualization.md): Библиотеки matplotlib и plotly (интерактивные графики). Рисуем классические японские свечи.
- [1.4. Знакомство с MOEX AlgoPack](./1-4-moex-algo.md): Установка библиотеки moexalgo. Структура запросов.

## Результат модуля

Скрипт, который скачивает историю любой акции и строит красивый график.
