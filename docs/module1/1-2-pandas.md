# 1.2. Основы Pandas

Pandas - это мощная библиотека Python для работы с данными. В контексте алгоритмической торговли мы будем использовать её для обработки временных рядов и финансовых данных.

## Введение в Pandas

Основные структуры данных:
- **Series** - одномерный массив с индексами
- **DataFrame** - двумерная таблица с индексами строк и колонок

```python
import pandas as pd

# Создание Series
s = pd.Series([1, 3, 5, 6, 8])

# Создание DataFrame
df = pd.DataFrame({
    'A': [1, 2, 3],
    'B': [4, 5, 6]
})
```

## Работа с временными рядами

Для финансовых данных важно правильно работать с датами и временем:

```python
# Преобразование строки в datetime
date = pd.to_datetime('2023-01-01')

# Создание диапазона дат
dates = pd.date_range('2023-01-01', periods=5, freq='D')

# Установка индекса по времени
df.index = pd.to_datetime(df.index)
```

## Ресемплинг данных

Ресемплинг позволяет изменять частоту временного ряда. Это особенно полезно для преобразования минутных данных в часовые или дневные.

### Пример: минутные свечи в часовые

```python
# Предположим, у нас есть DataFrame с минутными данными
# Индекс - datetime, колонки: open, high, low, close, volume

# Ресемплинг на часовые интервалы
hourly_data = df.resample('H').agg({
    'open': 'first',
    'high': 'max',
    'low': 'min',
    'close': 'last',
    'volume': 'sum'
})

print(hourly_data.head())
```

### Другие примеры ресэмплинга

```python
# Из минутных в 15-минутные
df_15min = df.resample('15T').last()

# Из часовых в дневные
df_daily = df.resample('D').last()

# Недельные данные
df_weekly = df.resample('W').last()
```

## Практическое применение

В алгоритмической торговле ресэмплинг используется для:
- Анализа данных на разных временных масштабах
- Создания мульти-таймфреймовых стратегий
- Оптимизации вычислений при работе с большими объемами данных

Освоив основы Pandas и ресэмплинга, вы сможете эффективно работать с любыми временными рядами финансовых данных.
