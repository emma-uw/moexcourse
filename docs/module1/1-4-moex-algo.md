# 1.4. Знакомство с MOEX AlgoPack

MOEX AlgoPack - это специализированная библиотека для работы с данными Московской биржи. Она предоставляет удобный интерфейс для получения исторических данных, котировок и другой рыночной информации.

![image](/img/content.png)

## Установка библиотеки

```bash
pip install moexalgo
```

Или через conda:

```bash
conda install -c conda-forge moexalgo
```

## Основные возможности

Библиотека предоставляет доступ к:
- Историческим данным по акциям и облигациям
- Данным по фьючерсам и опционам
- Котировкам в реальном времени
- Информации о торгах и индексах

## Структура запросов

### Получение исторических данных

```python
from moexalgo import Market, Ticker

# Создание объекта для работы с рынком акций
stocks = Market('stocks')

# Получение данных по конкретной акции
sber = Ticker('SBER')

# Скачивание исторических данных
data = sber.candles(date='2023-01-01', till='2023-12-31', period='D')
print(data.head())
```

### Структура данных

Библиотека возвращает данные в формате pandas DataFrame с колонками:
- `begin` - время начала свечи
- `open` - цена открытия
- `close` - цена закрытия
- `high` - максимальная цена
- `low` - минимальная цена
- `value` - объем торгов в рублях
- `volume` - количество бумаг

### Пример использования

```python
import pandas as pd
from moexalgo import Ticker

# Получаем данные по акции Газпром
gazp = Ticker('GAZP')

# Минутные данные за последний день
candles = gazp.candles(period='1m', limit=100)

# Преобразуем в DataFrame
df = pd.DataFrame(candles)
df['begin'] = pd.to_datetime(df['begin'])

print(f"Получено {len(df)} минутных свечей")
print(df.head())
```

## Особенности работы

- Данные возвращаются в обратном хронологическом порядке (свежие данные первыми)
- Максимальное количество записей за один запрос ограничено
- Для больших объемов данных используйте пагинацию
- Учитывайте лимиты API при частых запросах

Освоив основы работы с MOEX AlgoPack, вы сможете получать все необходимые данные для анализа и разработки торговых стратегий.
