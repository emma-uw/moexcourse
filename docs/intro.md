# Введение

Добро пожаловать в курс по алгоритмической торговле на Московской бирже (MOEX) с использованием API Finam! Этот курс предназначен для тех, кто хочет научиться создавать и управлять автоматическими торговыми стратегиями.

## Цели курса

- Освоить основы работы с финансовыми данными
- Изучить методы анализа рынка и генерации сигналов
- Разработать и протестировать торговые стратегии
- Реализовать системы для реального времени торговли
- Обеспечить безопасность и надежность торговых операций

## Структура курса

Курс разделен на 8 модулей, каждый из которых фокусируется на конкретном аспекте алгоритмической торговли:

1. **Настройка среды разработки** - установка необходимых инструментов
2. **Работа с рыночными данными** - анализ свечей и индикаторов
3. **Разработка стратегий** - идеи, сигналы и очистка данных
4. **Бэктестирование** - тестирование стратегий на исторических данных
5. **Тестирование и оптимизация** - метрики и улучшение производительности
6. **Системы реального времени** - асинхронное программирование и устойчивость
7. **Исполнение ордеров** - API, безопасность и типы ордеров
8. **Продакшн** - риски, логирование и развертывание

Приступим к изучению!
