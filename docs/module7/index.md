# Модуль 7: Интеграция с брокером (Execution)

Седьмой модуль курса посвящен интеграции торгового робота с брокерскими API. Мы научимся отправлять реальные ордера и управлять позициями.

## Цель

Совершить первую автоматическую сделку. Освоить работу с брокерскими API, обеспечить безопасность и надежность исполнения ордеров.

## Разделы

- [7.1. Выборы API](./7-1-api-choice.md): Обзор API российских брокеров (Finam Trade API, Tinkoff Invest API, Alor OpenAPI). Получение токенов доступа.
- [7.2. Безопасность](./7-2-security.md): Где хранить пароли и токены (файл .env)
- [7.3. Типы заявок](./7-3-order-types.md): Лимитные (Limit) vs Рыночные (Market) и др. Что лучше для алготрейдинга?
- [7.4. Обратная связь](./7-4-feedback.md): Получение статуса заявки (Исполнена / Отменена / Частично исполнена). Синхронизация позиции робота и счета.

## Результат модуля

Робот, торгующий в "Песочнице" (Sandbox) брокера или paper trading.
