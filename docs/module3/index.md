# Модуль 3: Поиск Альфы (проверка гипотез)

Третий модуль курса посвящен разработке торговых стратегий. Мы научимся формулировать гипотезы, проверять их на данных и создавать сигналы для автоматической торговли.

## Цель

Сформулировать торговую гипотезу и проверить ее математически. Научиться превращать идеи в код и сигналы.

## Разделы

- [3.1. Генерация идей](./3-1-ideas.md): Логика стратегий (Mean Reversion — возврат к среднему vs Trend Following — следование за трендом).
- [3.2. Корреляционный анализ](./3-2-correlation.md): Влияет ли дисбаланс объемов на цену через 5 минут? Строим матрицу корреляций (Heatmap).
- [3.3. Создание "Сигналов"](./3-3-signals.md): Программирование логики if/else. Пример: "Если дисбаланс > 80% и цена выше скользящей средней -> Сигнал 1".
- [3.4. Очистка данных](./3-4-cleaning.md): Работа с пропусками, выбросами и "битыми" данными.

## Результат модуля

Четко сформулированные правила стратегии на бумаге и в коде (функция generate_signals).
